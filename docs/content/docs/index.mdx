# CrowdSec Manager

CrowdSec Manager is a web interface for operating CrowdSec and Traefik-based remediation workflows.

## Start here

1. Read [Installation](/docs/installation) for the canonical setup.
2. Use [Quick Start](/docs/quick-start) for the fastest path.
3. Review [Configuration](/docs/configuration/environment) when you need custom paths or overrides.

## What this documentation covers

- Feature guides for dashboard, alerts, decisions, scenarios, captcha, logs, backups, and updates.
- Configuration guides for environment variables, networking, and volume mapping.
- API reference for core endpoints.
