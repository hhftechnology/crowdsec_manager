# Logs & Monitoring

Monitor your system with real-time logs and advanced analysis tools.

## Real-time Streaming
View live logs via a WebSocket connection. Logs are streamed directly from the container to your browser with minimal latency.

## Multi-service Support
Access logs for all integrated services:
- **CrowdSec**: Security alerts and decisions.
- **Traefik**: Access logs and routing errors.
- **Pangolin**: Manager application logs.
- **Gerbil**: Helper service logs.

## Advanced Analysis
Analyze Traefik logs with detailed statistics. View:
- Top client IPs.
- Most accessed paths.
- Status code distribution (2xx, 4xx, 5xx).
- User agent breakdown.

## Usage
1. Navigate to the **Logs** section.
2. Select the service you want to monitor from the dropdown.
3. Click **Start Stream** to begin viewing live logs.
4. Use the **Pause** and **Clear** buttons to manage the view.
